# Compose Tetris

A modern Tetris game built with **Jetpack Compose**, featuring a retro LED aesthetic, immersive sound effects, and intuitive controls.

## 🚀 Features

- **Retro LED Visuals**: Classic gameboy-style LED display and typography.
- **Immersive Audio**: Responsive sound effects for moving, rotating, and clearing lines.
- **Intuitive Controls**: Easy-to-use buttons for all actions, including a "Drop" feature.
- **Dynamic Difficulty**: Game speed increases as you clear more lines and level up.
- **Pause/Resume**: Save your progress and come back anytime.
- **Mute Toggle**: Play with or without sound.

## 📦 Getting Started

### Prerequisites

- Android Studio Flamingo or newer.
- Android SDK 33 or newer.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/TetrisGame.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on your emulator or physical device.
